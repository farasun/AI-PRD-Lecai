# 乐才成果社团赛事BRD

## 文档基础
基于「赛事模式等事宜讨论」会议记录核心内容梳理，融合**社团团长授权自主创建赛事活动**的新增业务需求，聚焦赛事B模式核心功能要求，仅明确业务落地标准，不涉及产品技术实现细节。

## 一、项目背景
为满足商家/赞助商/合作方通过赛事实现招生、引流的核心需求，同时赋能平台社团团长自主运营赛事，强化社团自驱力与平台「社团+赛事」的联动性，丰富平台赛事玩法，参考卡哇伊赛事运营逻辑，规划落地赛事B模式，支持**平台后台代建**和**团长授权自建**两种赛事搭建形式，实现「入社方可参赛」的核心业务规则，让社团赛事运营更灵活、可落地。

## 二、核心模式定义
赛事B模式为**合作方/社团专属社团赛事模式**，由合作方/授权团长发起专属赛事，参赛主体需先加入该赛事对应的平台专属社团，方可完成赛事报名与作品投稿；赛事支持**平台后台代建**和**团长系统授权后自主创建**两种形式，核心助力合作方引流招生、赋能团长自主运营社团，提升平台社团活跃度。

## 三、核心功能要点（业务层面）
### （一）基础核心规则功能（原B模式核心要求）
1. **参赛身份前置校验**
    - 赛事报名/作品上传环节，增加「专属社团身份」校验功能；
    - 未加入对应专属社团的用户，需完成入社操作后，方可进入赛事报名/投稿流程，不开放个人报名通道。
2. **赛事与专属社团强关联**
    - 该模式赛事统一命名为「社团挑战赛」，明确赛事发起主体为平台专属社团；
    - 赛事页面与对应专属社团页面实现双向关联，参赛作品自动归属至对应专属社团下。
3. **参赛作品社团标识**
    - 投稿作品的展示卡片中，需明确显示所属「赛事专属社团」信息，区分个人参赛作品与B模式社团参赛作品。

### （二）团长自主创建赛事功能（新增融合需求）
1. **团长赛事创建授权与专属入口**
    - 系统新增「赛事创建授权」功能，仅向授权团长开放赛事创建权限；
    - 授权团长的社团操作端，区分「纪录类活动」与「赛事类活动」创建入口，团长可自主选择创建赛事活动。
2. **赛事创建核心信息填写项**
    - 团长创建赛事活动时，需填写3个核心必填项：①活动作品提交截止日；②作品点赞（投票）截止日；③奖项设置以及领奖说明。
3. **系统自动化流程控制**
    - 系统根据团长填写的**作品提交截止日**，自动关闭作品投稿通道，截止后不可再提交新作品；
    - 系统根据团长填写的**作品点赞（投票）截止日**，自动关闭投票通道，截止后锁定投票数据，不支持二次投票/点赞。
4. **自动化获奖标记与结果公示**
    - 投票通道关闭后，系统按**作品点赞（投票）数**结合团长设置的**奖项设置规则**，自动标记对应获奖作品；
    - 平台为该模式赛事配置「获奖公告」模块，自动同步获奖作品、奖项信息、领奖说明，支持赛事结果公示；
    - 赛事结束后自动生成并锁定人气排行榜，排行榜数据与对应专属社团绑定展示。

## 四、落地核心要求
1. **赛事搭建形式**：支持**平台后台代建**和**团长授权自建**两种形式，短期以平台代建为主，长期逐步开放团长授权自建功能，满足不同场景的赛事搭建需求。
2. **用户留存规则**：仅做「入社参赛」的前置要求，**不限制用户赛后退出社团**，无强制留存的功能要求。
3. **奖品与交付规则**：奖品筹备、发放均由赛事发起方（合作方/授权团长）负责，平台仅按发起方填写的信息做奖品、奖项、领奖说明公示，奖品交付对接由发起方通过微信群/平台客服完成，平台不参与奖品实际交付。
4. **系统规则适配**：平台赛事规则模块需支持该模式赛事的自定义配置，所有B模式赛事均默认**仅对加入对应专属社团的用户开放报名权限**，不开放个人报名通道。
5. **授权管理要求**：系统需建立团长赛事创建授权机制，可灵活配置授权对象，未授权团长仅可创建原纪录类活动，无赛事活动创建权限。
6. **活动类型区分**：社团操作端需清晰区分「纪录类活动」与「赛事类活动」，两类活动的创建、展示、管理入口相互独立，不混淆。

## 五、核心业务价值
1. 满足合作方通过赛事引流、招生的核心诉求，提升平台与商家/赞助商的合作粘性，拓展平台商业合作场景；
2. 赋能社团团长自主运营赛事，提升团长的社团运营积极性与自驱力，解决社团日常活跃度不足的问题；
3. 强化平台「社团+赛事」的核心玩法，提升社团的用户加入率与平台社团模块整体活跃度，避免社团「一次性创建、闲置运营」的问题；
4. 丰富平台赛事类型，支持平台代建与团长自建双形式，覆盖不同合作方、不同社团的赛事运营需求，让赛事玩法更灵活、落地性更强；
5. 实现赛事流程的自动化管控与获奖标记，减少平台人工运营成本，同时让赛事结果更透明、可追溯。
