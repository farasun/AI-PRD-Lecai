# 乐才银龄术语映射表 (Silver-Age Terminology Glossary) v1.0

> [!NOTE]
> **目标**：将互联网黑话转换为中老年（50-70岁）熟悉的、有集体荣誉感和社交温度的语言。

| 互联网术语 (Tech/Product Term) | 乐才银龄术语 (Silver UI Label) | 含义与情绪背景 |
| :--- | :--- | :--- |
| **挂载/代表社团 (Attach/Represent)** | **代表单位参报 / 集体报到** | 强调非个人行动，而是从属于一个集体的正式身份，增加仪式感。 |
| **投递/投稿 (Submit/Post)** | **作品选送 / 报送** | 模仿线下老年大学或群博会的语境，暗示作品是经过“选拔”和“推介”的，更有面子。 |
| **投票/点赞 (Like/Vote)** | **喝彩 / 助力 / 加油** | 将冷冰冰的数字竞争转化为温暖的邻里支持和情感互动。 |
| **人气榜 (Leaderboard)** | **社团风采录 / 荣誉墙** | 避开“排行榜”带来的输赢焦虑，强调“展示风采”和“获得荣誉”。 |
| **团长审核 (Audit/Review)** | **团长确认 / 组织选送** | 赋予团长“带教官”或“班长”的行政权力和成就感，强化 B 端动力。 |
| **运营活动 (Campaign/Event)** | **联谊赛 / 展演会** | 让参与者觉得这是一场邻里社交，而不是一个消耗时间的互联网活动。 |
| **用户画像 (User Profile)** | **个人主页 / 我的资历** | 去掉“画像”这种互联网监视感，强调其作为“资历”和“名片”的展示价值。 |

## 文案准则 (Copywriting Principles)
1. **不使用拼音缩写或纯英文**：如需使用，必须配中文注释。
2. **拒绝傲慢**：不要使用“你可能喜欢”这种带算法优越感的词，改用“同好推荐”。
3. **字号适配**：所有的 UI 标签设计时应预留更长的字符宽度，以容纳字号加粗加大的需求。
