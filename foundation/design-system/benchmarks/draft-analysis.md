# 视觉逆向工程报告: "学习强国"老年版设计基因提取

## 1. 设计目标与适用场景

本报告旨在从“学习强国”老年版中提取出普适性的适老化设计规律，并将其转化为可被 AI 重用的 **Design Tokens (设计变量)** 与 **组件模式**。这些变量将用于我们项目中的线框图和后续的高保真 UI 设计。

## 2. 核心视觉特征 (Visual Footprint)

综合适老化应用的行业标准与“学习强国”大字版的常见特征，我们提取了以下关键视觉维度的规律：

### 2.1 排版与空间规律 (Typography & Spacing)

*   **超大字号体系**：
    *   为了确保可读性，基准字号大幅提升。
    *   在我们的 CSS 变量中，可以映射为：`--font-size-base: 18px` 到 `20px`。
*   **高对比度与行高**：
    *   文字与背景对比度满足 WCAG AAA 标准。
    *   行高通常维持在 `1.5` 到 `1.6` 以上，避免文字堆叠。
*   **大点击区与间距**：
    *   按钮、列表项的可点击区域高度至少 `48px` - `56px`。
    *   组件之间的留白（Margin）显著增加，使用“以白当代”的方式分隔内容，减少物理细线的干扰。

### 2.2 组件与布局范式 (Components & Forms)

*   **大圆角卡片**：
    *   信息常被包裹在带有明显圆角的白色卡片中，形成视觉上的“安全感”。
    *   变量映射：`--border-radius-card: 12px` 或 `16px`。
*   **去除冗余分割线**：
    *   尽量避免列表中的 `1px` 实线分割，改用充足的内边距或浅灰色背景带（如 `#F5F5F5`）进行区隔。
*   **强烈的视觉层级**：
    *   页面的主要结构（如顶部导航区块、底部操作栏）可能带有轻微但明确的投影 (`box-shadow`)，或者使用品牌底色来区分。

### 2.3 色彩规律 (Colors)

*   **主色调引导**：
    *   通常会有一个非常明确的高亮色（如强国的标致红 `#E60012`）用于核心操作（如“提交”、“签到”）。
*   **层次分明的灰度**：
    *   文字颜色并非纯黑，而是使用深灰（如 `#333333`）作为主文本，中灰（如 `#666666` 或 `#999999`）作为辅助说明文本。

## 3. 落地建议：Tailwind CSS 的变量化方案

为了支持 AI 准确套用，我们将这些规律转化为具体的 Tailwind CSS 扩展配置或是通用的 CSS 变量集。

### 示例 CSS 变量定义 (`foundation/design-system/style-guide/variables.css` 草案)

```css
:root {
  /* 品牌色 */
  --color-primary: #E60012; /* 这里未来可以替换为乐才的主色，比如特定的蓝或红 */
  --color-bg-base: #F5F7FA;
  
  /* 文本灰度 */
  --color-text-title: #222222;
  --color-text-body: #444444;
  --color-text-muted: #888888;

  /* 字号阶梯 (大字版) */
  --text-size-sm: 16px;
  --text-size-base: 18px;
  --text-size-lg: 20px;
  --text-size-xl: 24px;
  --text-size-xxl: 28px;

  /* 圆角与空间 */
  --radius-card: 16px;
  --radius-button: 8px;
  --spacing-element: 16px; /* 元素内间距 */
  --spacing-section: 24px; /* 模块间距 */
}
```
