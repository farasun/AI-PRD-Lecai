# CC-5 社团活动详情

## 状态 I：正常态（主界面）

### 模块：页头导航栏
- **[返回入口]**：显示“返回社团”文本，引导回流至 [CC-4 社团主页]。
- **[页面标题]**：显示当前业务节点名称“活动”。
- **[视觉降噪]**：**移除**旧版左上角的通用占位图标。

### 模块：活动信息概览区
本次无变化

### 模块：作品瀑布流列表
- **[布局排布]**：采用双列瀑布流卡片模型。
- **[卡片层级]**：作品标题 > 发布者昵称 > 互动底座。
- **[名称描述]**：发布者区域仅展示“昵称”，**严禁**携带“作者：”字符前缀。
- **[互动降噪]**：移除评论标签及评论数值，降低视觉负载。
- **[点赞强化]**：保留点赞图标与数值，并通过设计对比度予以突出显示。
- **[跳转逻辑]**：点击卡片主体进入 [CC-6 社团作品详情]。

### 模块：底部全局操作栏
- **[核心按钮]**：固定于屏幕底部。名称定义为：**“上传照片/视频”**。
- **[交互响应]**：点击后直接激活媒体类型选择流程（状态 III）。

### 页面通用逻辑
本次无变化

## 状态 II：空状态
本次无变化

## 状态 III：类型选择弹窗
- **[功能定义]**：作为流程中置节点。提供“照片 (最多9张)”与“视频 (限1个)”的分类入口。
- **[分发逻辑]**：点击选项后唤起对应的系统原生拾取器。

## 状态 IV：系统原生媒体选择项
- **[托管协议]**：由移动端操作系统承载预览与勾选逻辑。
- **[确认响应]**：完成媒体选择后，流程自动推进至 [CC-7 社团提交作品表单]。
