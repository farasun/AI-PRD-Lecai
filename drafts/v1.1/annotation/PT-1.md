# PT-1 我的积分

## 状态 I：有记录态

### 模块：顶部导航

- 返回操作：点击后返回 CC-3 主导航-我的
- 页面标题：「我的积分」

### 模块：积分余额展示区

- **区域标签**：「当前积分」
- **余额数字**：当前用户积分总余额，整数展示，从接口实时获取
- **单位**：「积分」
- **兑换指引入口（v1.1 增量）**：点击弹出「线下兑换说明弹层」（见状态 V）。
- **数据来源**：用户积分账户余额接口，进入页面时加载，下拉刷新可触发重新请求

### 模块：积分记录列表

- **排序规则**：按积分到账时间倒序排列（最新记录在最上方）
- **分页规则**：首次加载最新 20 条；上拉至底部触发加载更多，每次追加 20 条
- **每条记录字段**：
  - 来源说明：口语化文案，对应关系见下方枚举
  - 积分数额：格式「+N」，N 为整数
  - 时间戳：格式「MM-DD」（当年记录）
- **来源说明文案枚举**：

  | 业务行为 | 来源说明文案 |
  |:--|:--|
  | 每日签到（R1） | 每日签到 |
  | 上传社团活动作品（R2） | 上传作品至「{活动名称}」 |
  | 首次发布作品奖励（R3） | 首次发布作品（新手奖励） |
  | 点赞社团作品（R4） | 点赞社团作品 |
  | 评论社团作品（R5） | 评论社团作品 |
  | 加入社团（R6） | 加入社团 |
  | 首次加入社团奖励（R7） | 首次加入社团（新手奖励） |
  | 创建社团（R8，仅社团主） | 创建社团 |
  | 发布活动（R9，仅社团主） | 发布活动 |
  | 社团团员数达标（R10，仅社团主） | 社团成员达到 {N} 人 |
  | 社团作品数达标（R11，仅社团主） | 社团作品达到 {N} 件 |

- **加载完毕**：底部展示「已显示全部记录」，不再触发加载
- **边界情况**：若活动已被删除，R2 的活动名称占位显示为「已删除的活动」

### 模块：「如何赚积分」入口

- 固定于页面底部，列表滚动时始终可见
- 入口文案：「如何赚积分」，带向下展开图标
- 点击：从页面底部弹出规则弹层（切换至状态 II）

---

## 状态 II：规则弹层展开态

### 模块：规则弹层

- **弹层形态**：从页面底部滑入，覆盖页面下半部分；弹层后方页面内容不可交互（遮罩层）
- **关闭方式**：
  - 点击右上角关闭图标
  - 点击弹层外遮罩区域
  - 两种方式效果相同：弹层滑出，返回状态 I
- **弹层标题**：「如何赚积分」

**第一层：核心规则（始终展示，3 条）**

- 条目 1：日历图标 + 「每天打卡签到，可获得 N 积分」（N 从配置表读取）
- 条目 2：图片图标 + 「上传作品到社团活动，可获得积分」
- 条目 3：团队图标 + 「加入一个社团，可获得 N 积分」（N 从配置表读取）

**展开入口**：「查看更多获积方式」，带向下图标，点击展开第二层（切换至状态 III）

---

## 状态 III：规则弹层 - 展开态

### 模块：规则弹层（展开）

- 在状态 II 的基础上，追加展示以下规则：
  - 「评论社团作品，每天最多 N 次，每次获得 N 积分」
  - 「点赞社团作品，每天最多 N 次，每次获得 N 积分」
  - 「首次发布作品，额外获得 N 积分（新手礼包）」
  - 「首次加入社团，额外获得 N 积分（新手礼包）」
  - 「社团主专属：创建社团、发布活动、社团达到里程碑，可获得额外积分」

- 展开入口文案切换为「收起」，带向上图标，点击收起第二层（回到状态 II）
- 弹层高度超出屏幕时，弹层内容区域可独立滚动

---

## 状态 IV：空记录态

### 模块：空状态提示

- 触发条件：用户积分记录数为 0（从未触发任何积分行为）
- 余额展示区正常展示（余额为 0）
- 记录列表区域替换为空状态展示：
  - 空状态图标：语义为「空箱」的插图
  - 说明文案：「还没有积分记录，去签到或上传作品开始累积吧！」
  - 引导按钮：主操作按钮，文案「去签到」，点击跳转 PT-2 每日签到
- 底部「如何赚积分」入口正常展示，逻辑不变

---

## 状态 V：线下兑换说明弹层

### 模块：兑换指引 Modal

- **弹层形态**：屏幕居中的全局模态框（Modal），带有深色模糊遮罩，阻断底层内容交互。
- **动态文案获取**：无接口请求，纯本地静态文案展示。
- **说明文案**：「线上积分商城正在紧张筹备中。目前，您可以凭借积分，向您的**社团主（班长）**报名兑换线下活动的专属福利或精美纪念品。」
- **关闭机制**：
  - 点击底部主按钮「我知道了」
  - 点击弹框外围背景遮罩
  - 以上操作皆关闭该模态框，无刷新返回状态 I（主界面）。
