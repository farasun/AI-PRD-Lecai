# CC-31 赛事管理-Web端

## 状态 I：配置管理主视图

### 模块：顶层中控开关
*   **Dimension 1 (Phase)**：赛事阶段状态机切换。修改后全量生效于 C 端和团长端面板。
*   **Dimension 2 (Audit)**：审核模式全局开关。ON: CC-7 投稿需 CC-17 审核；OFF: 投稿即公开。
*   **Dimension 3 (Interaction)**：点赞/评论防刷开关。开启后 CC-6 锁定。
*   **Dimension 4 (Publicity)**：公示控制及富文本配置。

### 模块：效果实时预览 (Right Panel)
*   **逻辑定义**：嵌入移动端 Simulator。
*   **同步机制**：当管理员修改上述开关时，右侧预览框中的 UI 状态按逻辑映射规则即时演变（模拟 CC-26 行为）。

## 状态 II：状态变更确认 (Modal)

### 模块：二次确认对话框
*   **触发条件**：点击 [发布全局配置]。
*   **数据内容**：
    *   变更项差异对比。
    *   受影响用户数预估记录。
*   **交互规则**：需手动确认“我已知晓全网状态将同步更新”复选框。
