## 四、核心业务流程（三条主线）

**主线 A：师训认证流程**
```
加入社团（CC-9/现有流程）
  → 查看认证类活动（CC-5，成员可见）
  → 并行：提交作品（CC-7，认证模式）+ 填写社团问卷（CC-23）
  → 两项完成 → 等待评审
  → 认证通过 → CC-24 证书展示 → 分享到微信
```

**主线 B：活动权限配置流程（社团主侧）**
```
CC-12 活动管理
  → 创建/编辑活动
  → 进入 CC-12-1 权限设置
  → 选择可见范围（公开 / 成员 / 白名单）
  → 保存，活动按权限展示
```

**主线 C：赛事报名（含引导加入社团）**
```
CH-1 赛事详情
  → 点击报名
  → [未加入关联社团] 引导弹层：一键加入专属社团
  → 加入成功 → 完成赛事报名
  → 在活动页提交参赛作品（CC-5 → CC-7）
```
