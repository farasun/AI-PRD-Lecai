## 四、核心业务流程（三条主线）

**主线 A：社团轻量工具（师训认证）**
```
加入卡哇伊社团（CC-9 现有流程）
  → 查看活动下课程资料（CC-5，外链跳转卡哇伊视频）
  → 提交认证作品（CC-7，选「认证提交」标签）
  → 填写教学心得问卷（CC-23，≤5 题，辅助性）
  → 两项完成 → 卡哇伊评审
  → 认证通过 → CC-6 作品详情 → 复用证书弹层展示证书 → 分享微信
```

**主线 B：活动权限配置（社团主侧）**
```
CC-12 活动管理
  → 创建/编辑培训类活动
  → 进入 CC-12-1 设置「仅社团成员可见」
  → [可选] 进入 CC-12-2 挂载「教学心得问卷」
  → 保存，活动按权限展示
```

**模块一·路径1：从社团发现赛事**
```
CC-4 社团主页「关联赛事」区块
  → 点击赛事卡片 → CH-1 赛事详情
  → [已加入社团] 直接报名
  → [未加入] 入团引导弹层 → 加入 → 自动完成报名
```

**模块一·路径2：从赛事引导入团**
```
CH-1 赛事详情
  → 点击「参赛报名」
  → [未加入关联社团] 引导弹层：一键加入专属社团
  → 加入成功 → 自动完成赛事报名
```
