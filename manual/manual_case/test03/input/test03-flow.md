# test03 流程需求

请参照 Main_PageList.md 列出本次手册的页面节点...
1、首页CC-1，点击社团管理
2、进入社团管理CC-10，逐个介绍功能
3、先是CC-11社团信息编辑，可以改名称、头像等
4、然后是CC-18，社团权限设置，可以控制加入、投稿是不是要审核，建议按默认
5、CC-16，成员加入审核，如开通审核可以在次操作
6、CC-17，作品审核，如开通审核可以在次操作
7、此处还能看到社团的所有作品，CC-17已处理tab状态。
8、点击进入作品详情页，CC-5，社团长可以看到右下角的下架按钮
9、可以点击对作品进行下架，下架后就不公开了，CC-5 下架弹窗
10、CC-16-1，查看全部成员
11、CC-12，管理社团活动，这是主要功能，需要仔细讲
12、CC-12点击底部案件，弹出CC-12的新建活动弹层，输入名称等提交
13、CC-12，可以对活动进行置顶操作，见操作弹层
14、CC-12的排序界面，可以修改活动的上下展示顺序
15、回到CC-4，社团首页。新建的活动没有作品时，在社团首先不展示
16、需要上传，选择刚刚新的活动， CC -7弹层选择界面
17、上传完成作品回到CC-4，就能看见到新活动了
18、在活动点击底部的上传按钮，可以直接上传作品到活动，CC-7活动锁定状态
19、活动内上传完成作品，回到CC-5活动详情页
20、可以进入活动内的作品详情页CC-6，进行作品分享（分享弹层）
