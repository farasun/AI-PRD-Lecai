# CC-5 社团活动详情

## 状态 I：正常态（主界面）

### 模块：页头导航栏

**显示与约束**
- **[返回入口]**：包含“返回社团”文本提示。
- **[页面标题]**：显示当前所属业务模块名称“活动”。
- **[更多菜单]**：提供次级功能操作入口。

**交互与跳转**
- 点击 **[返回入口]**：返回至 `CC-4 社团主页`。
- 点击 **[更多菜单]**：唤起功能选项组。

### 模块：活动信息概览区

**显示与约束**
- **[活动名称]**：动态获取当前活动标题。
- **[统计指标-作品数]**：展示当前活动关联的已发布作品总数。
- **[统计指标-获赞数]**：展示活动下所有作品累计获得的赞数。
- **[搜索入口]**：提供模糊检索功能。**占位提示**：“按作品名、作者名搜索”。
- **[排序切换]**：显示当前生效的排序规则（默认：最新发布）。

**交互与跳转**
- 点击 **[搜索入口]**：进入搜索输入模式。
- 点击 **[排序切换]**：呼出排序维度选择。

### 模块：作品瀑布流列表

**显示与约束**
- **[作品项]**：双列交错列表展示。
- **[作品标题]**：**展示逻辑**：若数据中标题字段为空，则在 UI 上隐藏该文本占位，不保留空白。
- **[发布者昵称]**：展示作品作者名称。
- **[点赞数]**：展示单个作品的获赞数值，配套点赞状态图标。

**交互与跳转**
- 点击 **[作品项主体]**：跳转至 `CC-6 社团作品详情`。
- 点击 **[点赞图标]**：触发点赞逻辑判定（点赞/取消）。

### 模块：底部全局操作栏

**显示与约束**
- **[操作入口]**：固定于视口底部。
- **[核心动作]**：文案定义为“上传照片/视频”。符合高亮操作逻辑。

**交互与跳转**
- 点击 **[核心动作]**：唤起 `状态 III：类型选择弹窗`。

### 页面通用逻辑

**数据规范**
- **初始排序**：服务端接口默认按 `publish_time`逆序返回。
- **分页加载**：支持列表滚动增量加载。

**异常处理**
- **无数据状态**：若当前活动无作品记录，UI 自动切换至 `状态 II：空状态`。

## 状态 II：空状态

**显示与差异点**
- **[状态引导]**：展示业务占位图形。
- **[提示文案]**：文案为“这里还在虚位以待，请您来开个头吧！”。
- **[首发动作按钮]**：引导用户进行首次发布的强逻辑按钮。

**交互跟随**
- 点击 **[首发动作按钮]**：唤起 `状态 III：类型选择弹窗`。

## 状态 III：类型选择弹窗

### 模块：媒体类型筛选面板

**显示与约束**
- **[背景遮罩]**：主界面覆盖半透明层以聚焦操作。
- **[选项-照片]**：提供单选入口。文案定义为“上传照片 (最多9张)”。
- **[选项-视频]**：提供单选入口。文案定义为“上传视频 (1个，100MB以内)”。
- **[动作-取消]**：提供显性的关闭/回退入口。

**交互与跳转**
- 点击 **[选项-照片]**：跳转至 `状态 IV：系统原生媒体选择项`，并激活照片过滤模式。
- 点击 **[选项-视频]**：跳转至 `状态 IV：系统原生媒体选择项`，并激活视频过滤模式。
- 点击 **[动作-取消/背景遮罩]**：收起弹窗，返回主界面当前位置。

## 状态 IV：系统原生媒体选择项

### 模块：系统托管选择器

**显示与约束**
- **容器属性**：此模块由操作系统（iOS/Android）原生托管，UI 表现随设备系统版本异构。
- **过滤机制**：根据前置触发类型（照片/视频）自动筛选对应的媒体文件格式。
- **选择上限**：
    - 照片模式：允许最多勾选 9 个文件。
    - 视频模式：仅允许勾选 1 个文件。

**交互与跳转**
- **确认选择**：携带已选媒体数据，自动跳转至 `CC-7 社团提交作品表单`。
- **取消选择**：返回 `CC-5` 主界面，并默认收起所有临时弹层。
